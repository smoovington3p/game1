# Game1

Hybrid-casual block puzzle (iOS / Android)

## Overview

Game1 is a mobile block puzzle game targeting iOS and Android platforms. Built with Unity for cross-platform deployment.

## Setup in Unity Hub

1. Open Unity Hub
2. Click **Open** (or **Add**)
3. Navigate to this folder and select it directly
4. Unity Hub will detect the project and open it in the appropriate Unity version
